What is GlobalSphere?

GlobalSphere is a Minecraft server for both Bedrock and Java editions. It uses a map that includes Asia, parts of Africa, a sliver of Europe, and much of Oceania. The server itself runs on 1.20.4, and we will update to the latest version once all resources we use are stable (and we support the latest Bedrock edition). The server theme is set to the 1890's.

The server itself is host to a friendly community of players that build great things. The showcase on our homepage is perfect evidence of the great building community here.

Features

GlobalSphere has various features, such as brewing (recently added), movecraft (ships, planes, elevators, etc.), several items such as totems of undying and ender crystals being restricted, and a plugin built by our developer for new players.
