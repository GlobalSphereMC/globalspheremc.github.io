What's up at GlobalSphere?

December 4th, 2022.

GlobalSphere is now launching textures for brewery!

The brewery plugin now has custom textures for each brew and quality. These are available on both Java and Bedrock editions.

To get the textures, make sure to download the resource pack. If you do not get a prompt to download it on Java edition, make sure you have "Server resource packs" set to "prompt" in your server settings!
